Detailed Description

template<class TypeReal>
class gmath::Matrix4< TypeReal >

Matrix class (3x3). [Xx, Xy, Xz] [Yx, Yy, Yz] [Zx, Zy, Zz]

This class represents a 3x3 matrix that can be used to store rotations transformations. This matrix is ROW MAJOR.
