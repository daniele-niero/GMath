Matrix class (4x4). [Xx, Xy, Xz, Xw,] [Yx, Yy, Yz, Yw,] [Zx, Zy, Zz, Zw,] [Px, Py, Pz, Pw]

This class represents a 4x4 matrix that can be used to store transformations. This matrix is ROW MAJOR.
