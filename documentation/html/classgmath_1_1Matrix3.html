Detailed Description

template<class TypeReal>
class gmath::Matrix3< TypeReal >

Matrix class (3x3).
[Xx, Xy, Xz]
[Yx, Yy, Yz]
[Zx, Zy, Zz]

This class represents a 3x3 matrix that can be used to store rotations transformations.

Note:
This matrix is ROW MAJOR.

           The generated rotation matrix will rotate the vector _from into
           the vector to. _from and to must be unit vectors!
